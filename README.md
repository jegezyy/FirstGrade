# 📚 Первая Зачётка

> Веб-приложение для продажи восстановленной техники с системой заказов и авторизацией

---

## 📌 О проекте

**Первая Зачётка** — это веб-приложение на Flask, которое позволяет пользователям покупать восстановленную технику (ноутбуки, принтеры и др.), а администраторам — управлять товарами и заказами.

Проект реализует базовую логику маркетплейса:
- каталог товаров
- регистрация и авторизация
- оформление заказов
- админ-панель

---

## 🔥 Актуальность

Проект демонстрирует:
- работу с базой данных (SQLite)
- систему аутентификации (JWT + cookies)
- разграничение прав доступа
- CRUD-операции
- серверный рендеринг (Flask + Jinja2)

---

## ⚙️ Основной функционал

### 🛍️ Каталог товаров
- Просмотр всех товаров
- Фильтрация по:
  - типу (`type`)
  - состоянию (`condition`)
- Сортировка по цене
- Отображение только активных товаров

---

### 👤 Пользователь

#### Регистрация и вход
- Регистрация с проверкой пароля
- Хеширование пароля
- Авторизация через JWT (cookie)

#### Профиль
- Просмотр профиля по username
- Обновление токена

---

### 🛒 Заказы
- Покупка товара
- Проверка на повторную покупку
- Автоматическое изменение статуса товара (`sold`)
- Просмотр своих заказов

---

### 🔐 Роли

#### Пользователь
- Просмотр товаров
- Покупка
- Просмотр заказов

#### Администратор
- Создание товара
- Редактирование товара
- Удаление товара
- Просмотр всех заказов
- Изменение статуса заказа
- Доступ к dashboard

---

### 📊 Dashboard (админ)

API `/api/dashboard`:
- Статистика заказов по статусам
- Количество товаров по типам
- Заказы по датам

---

### 🧪 API

- `/api/about` — данные о проекте из JSON  
- `/api/hash/<text>` — генерация hash + логирование

## 🗄️ Структура базы данных

### `user`
```sql
id INTEGER PRIMARY KEY
username TEXT UNIQUE
email TEXT UNIQUE
password TEXT
is_admin INTEGER
uuid TEXT
created_at TEXT
```

### `item`
```sql
id INTEGER PRIMARY KEY
title TEXT
price INTEGER
type TEXT
condition TEXT
description TEXT
number TEXT
status TEXT
```

### `orders`
```sql
id INTEGER PRIMARY KEY
user_id INTEGER
item_id INTEGER
status TEXT
created_at TEXT
```

### `hash_log`
```sql
id INTEGER PRIMARY KEY
request TEXT
result TEXT
created_at TEXT
```

## 🛠️ Технологический стек

- **Backend:** Flask, FastAPI  
- **База данных:** SQLite(временно)  
- **Аутентификация:** JWT  
- **Шаблоны:** Jinja2  
- **Безопасность:** werkzeug.security  

---

## 🔐 Безопасность

- Хеширование паролей  
- JWT с временем жизни  
- Защищённые маршруты:
  - `@jwt_required`
  - `@admin_required`

---

## 🚀 Запуск

```bash
pip install -r requirements.txt
python main.py
```

Открыть в браузере:
```
http://127.0.0.1:5000
```
