# 📚 Первая Зачётка

> Маркетплейс восстановленной техники для абитуриентов и студентов

---

## 📌 О проекте

**Первая Зачётка** — это онлайн-платформа, где абитуриенты и студенты могут купить качественную восстановленную технику (ноутбуки, принтеры, планшеты) по цене в 2–3 раза ниже рыночной. Мастерские по ремонту оргтехники размещают проверенные устройства на платформе, а покупатели получают гарантию и честную историю каждого товара.

---

## 🔥 Актуальность и проблема

Каждый год сотни тысяч абитуриентов поступают в вузы с ограниченным бюджетом. Новый ноутбук — это 50 000–100 000 ₽, которых у большинства студентов нет. При этом мастерские по ремонту оргтехники ежедневно восстанавливают десятки устройств, которые сложно продать без готового канала сбыта.

**Проблема №1 — у студента:** нет денег на новую технику, а б/у на Avito — кот в мешке без гарантии.  
**Проблема №2 — у мастерской:** восстановленная техника лежит на складе, покупателей искать некогда.

**Первая Зачётка** соединяет их напрямую — с проверкой, гарантией и удобным сервисом.

## ⚙️ Ключевой функционал

### 🛍️ Каталог техники
- Фильтрация по типу устройства, цене, состоянию и гарантии
- Карточка товара с фото, историей ремонта и оценкой мастера

### 👤 Личный кабинет
- **Для покупателя:** история заказов, список желаемого, гарантийные обращения, бонусная система
- **Для мастерской:** витрина товаров, статистика продаж, выплаты
- Система отзывов и рейтингов

### 📝 Блог для абитуриента
- Статьи: «Какой ноутбук нужен на твою специальность»
- Гайды: «Как проверить б/у ноутбук перед покупкой»
- Чеклисты: «Что нужно распечатать для поступления»

### 💰 Дополнительно
- Онлайн-оплата через ЮKassa / Tinkoff
- Программа «Сдай старое — получи скидку»

## 🗄️ Структура базы данных

```
users
├── id (PK)
├── email
├── password_hash
├── full_name
├── phone
└── created_at

products
├── id (PK)
├── title
├── category (laptop | printer | tablet | other)
├── price
├── condition (good | excellent | like_new)
├── warranty_days
├── repair_history (text)
├── is_verified (bool)
├── status (active | sold | archived)
└── created_at

product_images
├── id (PK)
├── product_id (FK → products)
├── image_url
└── is_primary (bool)

orders
├── id (PK)
├── buyer_id (FK → users)
├── product_id (FK → products)
├── status (pending | paid | shipped | delivered | cancelled)
├── total_price
├── delivery_address
└── created_at

reviews
├── id (PK)
├── order_id (FK → orders)
├── user_id (FK → users)
├── rating (1–5)
├── comment
└── created_at

blog_posts
├── id (PK)
├── title
├── slug
├── content
├── tags
└── published_at
```
